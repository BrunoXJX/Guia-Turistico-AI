# VOYAGE AI

Protótipo mobile-first de um guia turístico inteligente em PT-PT.

## Stack

- React + Vite
- TypeScript
- Tailwind CSS
- React Router
- Lucide Icons

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Rotas

O deploy usa hash routing para funcionar bem em hosting estático:

- `/#/`
- `/#/tours/vilnius-segredos`
- `/#/audio/vilnius-segredos`
- `/#/itinerario`
- `/#/voz`
- `/#/passaporte`
- `/#/perfil`
